# 🏡 구해줘 홈즈 Web Vue 관통 프로젝트
![스크린샷_2022-11-16_오후_11.10.42](/uploads/f21fd5a202ddedc6f382e5f3deed0c72/스크린샷_2022-11-16_오후_11.10.42.png)

## ✨ 프로젝트 목표

* 웹 아키텍처를 이해하고 활용할 수 있다.
* Vue.js를 활용하여 SPA 웹 서버를 구축할 수 있다.
* MVC 아키텍처를 활용하여 확장성과 유지보수성 좋은 웹 서버를 구축할 수 있다
